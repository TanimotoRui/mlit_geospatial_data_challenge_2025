# データ説明

## 概要

第2回 国土交通省 地理空間情報データチャレンジでは、以下のデータが提供される：

1. **国土数値情報**（国土交通省）
2. **登記所備付地図データ**（法務省）
3. **物件データ**（株式会社LIFULL）

## 1. 国土数値情報

全国の地理空間情報を含むGISデータ。約190項目から構成される。

### 主な項目（想定）
- 行政区域
- 用途地域
- 土地利用
- 公共施設（学校、病院など）
- 交通機関（駅、バス停など）
- その他の地理空間情報

### データ形式
- シェープファイル（.shp）またはGeoJSON等のGIS形式

## 2. 登記所備付地図データ

不動産登記に関連する地図や図面の電子データ。

### 主な項目（想定）
- 地番情報
- 境界情報
- その他登記関連情報

## 3. 物件データ（LIFULL）

### 対象期間
- 2019年〜2023年の1月、7月

### 物件タイプ
- 中古マンション
- 中古戸建

### 主な項目（想定）
- **目的変数**: 売買価格
- **位置情報**: 緯度経度、住所など
- **物件情報**: 
  - 面積（専有面積、土地面積など）
  - 築年数
  - 間取り
  - 階数
  - 構造
  - その他物件属性

## データの配置

```
data/
├── raw/
│   ├── kokudo_suuchi/      # 国土数値情報
│   ├── touki_data/         # 登記所備付地図データ
│   └── lifull/             # LIFULL物件データ
│       ├── train.csv       # 学習データ
│       └── test.csv        # テストデータ
```

## 注意事項

- データの詳細仕様は、SIGNATE上の「データREADME.pdf」を参照
- データの取り扱いは利用規約に従うこと
- 外部データの追加利用可否は規約を確認すること
